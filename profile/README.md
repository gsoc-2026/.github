# GSoC '26

this repository is where i’m documenting seriously for GSoC 2026.

right now I’m spending time understanding and contributing to the following projects:

- google-deepmind/gemma  
- google-deepmind/torax  
- google-deepmind/formal-conjectures  
- google-deepmind/jax_privacy  
- google-deepmind/alphafold3  

i’m reading the source, tracing execution paths, understanding design decisions, and identifying areas where I can contribute meaningfully.

this repo documents that process. i keep track of the pull requests I open (and the reasoning behind them), small prototypes i build to understand specific components, proposal drafts, benchmark experiments, and notes on parts of the system that took time to fully grasp. if something felt inefficient, i test alternatives. If something was elegant, I try to understand why it works.

my interests going into gsoc '26 sit around large-model training, scientific computation, formal reasoning, and computational biology.

few organisations that i'm interested for proposals are:
- open-genome-informatics (https://gmod.org/wiki/GSOC_Project_Ideas_2026),(https://summerofcode.withgoogle.com/programs/2026/organizations/open-genome-informatics)
- CERN-HSF (https://hepsoftwarefoundation.org/gsoc/2026/summary.html), (https://summerofcode.withgoogle.com/programs/2026/organizations/cern-hsf)
- DeepChem (https://summerofcode.withgoogle.com/programs/2026/organizations/deepchem)
- ML4Sci (https://summerofcode.withgoogle.com/programs/2026/organizations/machine-learning-for-science-ml4sci)

this isn’t a polished showcase. It’s a working log of what I’m building, testing, and proposing.

if you’re here, you’ll see the progression in real time.
